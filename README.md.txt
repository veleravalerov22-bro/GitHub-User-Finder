# 🐙 GitHub User Finder - Поиск пользователей GitHub

**Автор:** Воронков Никита  
**Версия:** 1.0  
**Дата:** Май 2026

## 📋 Описание программы

**GitHub User Finder** — это графическое приложение для поиска пользователей GitHub через официальное API. Программа позволяет:

- 🔍 **Поиск пользователей** по имени или логину
- ⭐ **Добавление в избранное** понравившихся пользователей
- 💾 **Сохранение избранных** в JSON файл
- 📊 **Просмотр детальной информации** о пользователе (репозитории, подписчики и т.д.)
- 🌐 **Быстрый переход** на профиль GitHub в браузере

## 🚀 Инструкция по запуску

### Требования
- **Python 3.8** или выше
- Tkinter (входит в стандартную поставку Python)
- Интернет-соединение для работы с GitHub API

### Запуск приложения

```bash
# 1. Клонирование репозитория
git clone https://github.com/voronkov-nikita/github-user-finder.git
cd github-user-finder

# 2. Запуск программы
python main.py
